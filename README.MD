#menke


## 技术about
    ● 核心框架：Spring Boot 1.5.1.RELEASE
    ● 安全框架：Spring Security
    ● 任务调度：Spring + Quartz
    ● 持久层框架：MyBatis 3.4 + MyBatis-Plus 2.0+
    ● 数据库版本控制：Liquibase
    ● 数据库连接池：Alibaba Druid 1.0
    ● 缓存框架：Redis(后续改成多种缓存切换)
    ● 日志管理：logback
    ● 前端框架：Bootstrap + Jquery
